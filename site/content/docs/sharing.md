Trove lets you share individual files or entire folders with anyone — no account required to access a share.

## File shares

To share a file, open it and click **Share**. You can configure:

- **Password** — optional, required to access the link
- **Expiry** — the link stops working after a set date
- **Download limit** — the link expires after N downloads

The share URL can be sent to anyone. Recipients can view and download the file directly in their browser.

## Folder shares

Folder shares work the same way and expose all files under that folder path (including subfolders). Navigate to a folder, click **Share folder**, and configure the same options.

Recipients see a read-only file browser for the shared folder. They can browse subdirectories and download individual files.

## Managing shares

Your active shares are listed under **Settings → Shares**. From there you can:

- See share URLs
- Edit expiry or download limits
- Revoke a share at any time

Revoking a share immediately invalidates the link — any subsequent access attempt returns a 404.

## Password-protected shares

When a share has a password set, recipients are shown a password prompt before gaining access. The session is valid for the browser tab — closing and reopening requires re-entering the password.

Passwords are hashed and never stored in plain text.
